# Anupam Paikade - Portfolio

A premium, high-conversion personal portfolio website designed for a Product Management aspirant.
Built with React, Vite, Tailwind CSS, and Framer Motion.

## 🚀 How to Run Locally

Currently, Node.js is required to run this project, which seems to be missing on your system.
Please follow these steps to get your portfolio up and running:

1. **Install Node.js:**
   Download and install Node.js (LTS version) from the official website: [https://nodejs.org/](https://nodejs.org/)

2. **Open Terminal (or PowerShell/Command Prompt):**
   Navigate to this folder (`c:\Users\ANUPAM\Downloads\Anupam Portfolio`).

3. **Install Dependencies:**
   Run the following command in your terminal to install all required packages:
   ```bash
   npm install
   ```

4. **Start the Development Server:**
   Run the following command to start viewing your portfolio:
   ```bash
   npm run dev
   ```

5. **View the Website:**
   Open your browser and navigate to `http://localhost:5173`.

## 🎨 Design Features
- Clean White & Black aesthetic with an Indigo Accent.
- Beautiful scroll animations (Framer Motion).
- Glassmorphism effects (Navbar & Cards).
- Responsive on all devices.
